# ✈️ Airline Booking AI Assistant

An AI-powered airline customer support and booking assistant built using Streamlit and OpenRouter LLMs.

## Project Overview

This project was developed as part of my LLM Engineering learning journey.

The assistant can:

- Answer airline-related customer queries
- Assist users with flight booking conversations
- Provide travel and airline support information
- Maintain multi-turn conversations
- Simulate a real airline customer support experience

## Technologies Used

- Python
- Streamlit
- OpenRouter API
- Large Language Models (LLMs)
- Pandas
- NumPy

## RAG Experiments

During development, I also explored:

- Retrieval-Augmented Generation (RAG)
- ChromaDB Vector Database
- Embeddings and Semantic Search
- Airline FAQ and customer support datasets

The RAG implementation notebooks are included in this repository as part of my experimentation and learning process.
