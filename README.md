# Airline Booking AI Assistant

AI-powered airline customer support and ticket booking assistant built using Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), ChromaDB, LangChain, and OpenRouter.

## Project Overview

This project simulates an airline customer support chatbot that can:

- Answer airline-related customer queries
- Provide flight information
- Assist with ticket booking
- Retrieve information from a knowledge base using RAG
- Generate intelligent responses using LLMs

## Features

- Conversational AI chatbot
- Flight booking assistance
- Retrieval-Augmented Generation (RAG)
- Vector database search with ChromaDB
- OpenRouter LLM integration
- Prompt engineering and tool calling

## Technologies Used

- Python
- LangChain
- ChromaDB
- OpenRouter
- Hugging Face
- Vector Embeddings
- Jupyter Notebook
